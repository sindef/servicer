# Servicer v0.1.0 Release Notes

Release date: 2026-05-25

## Highlights

- First tagged production release for Servicer control-plane components.
- Tagged workflows publish multi-architecture images and a rendered install manifest.
- Release security workflow performs Trivy image scanning and keyless cosign signing.

## Published release artifacts

- `ghcr.io/sindef/servicer-manager:v0.1.0`
- `ghcr.io/sindef/servicer-bff:v0.1.0`
- `ghcr.io/sindef/servicer-web:v0.1.0`
- `ghcr.io/sindef/servicer-tools:v0.1.0`
- `ghcr.io/sindef/servicer-syncer:v0.1.0`
- `servicer-install-v0.1.0.yaml` (GitHub Release asset)

## Install

```bash
kubectl apply -f https://github.com/sindef/servicer/releases/download/v0.1.0/servicer-install-v0.1.0.yaml
```

## Notes

- Production install path uses `deploy/` and does not include the local-demo syncer sidecar.
- Preview product areas and remaining partial coverage are tracked in [Feature gap analysis](../feature-gaps.md).
